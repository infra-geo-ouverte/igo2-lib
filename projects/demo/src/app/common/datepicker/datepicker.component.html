Datepicker reference

Use calendarType to control what the user selects and how the date is displayed in the field. Use today and now when you want a relative value instead of a fixed date.

Option Use it for What the user selects Example Notes
calendarType="date" Standard date picking A full date 2026-05-29 Best choice for forms that need a day, month, and year.
calendarType="month" Month-based input A month within a year 2026-05 Useful for reporting periods, billing cycles, and monthly filters.
calendarType="year" Year-only input A year 2026 Useful when the day and month do not matter.
calendarType="datetime" Date and time workflows A date, with support for relative now 2026-05-29 14:30 Use this when the current moment matters more than only the current day.
today Relative date value The current day today This is a dynamic keyword, not a fixed calendar date.
now Relative datetime value The current date and time now Use this with calendarType="datetime".
