The icon is a core concept for many packages of the IGO2 library. The icon system is base on Google Font - Material Symbol who use font ligature to configure the icon

Getting started

Static icon font (Most light in term of data)

In your index.html file, add the variable font stylesheet request to your head tag and 
{{exampleIndexStaticHtml}}

Variable icon font (Alternative)

  1. In your index.html file, add the variable font stylesheet request to your head tag and 
    {{exampleIndexHtml}}
  2. Then add the current class with variable axes configuration to icons in your style.scss file. 
    {{exampleStyleVariable}}
  3. Finally, add the icon provider configuration in your main file main.ts 
    {{exampleProvider}}
